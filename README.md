# 前端 Agent 交互界面 — 智能旅游规划师原型

一个 **AI Agent 应用的高保真前端原型**，以「智能个人旅游一站式规划师」为场景，展示多 Agent 协作类产品的现代化交互界面设计。

## 界面亮点

- **多 Agent 任务流可视化**：不同 Agent（行程规划 / 交通 / 住宿 / 预算）以独立配色区分，任务状态一目了然
- **对话式指令输入**：自然语言输入区 + 历史会话侧边栏，还原主流 AI 产品交互范式
- **1440×1024 标准画布**：按桌面端设计稿尺寸精确布局，适合作为开发交付原型
- **纯 CSS 实现**：无任何框架依赖，使用 CSS 变量统一设计令牌（色彩 / 尺寸 / 圆角 / 阴影）

## 技术实现

- 单文件 `travel-planner-mockup.html`，零依赖，双击即可在浏览器打开
- CSS 自定义属性（Design Tokens）管理主题
- Flexbox 布局 + 系统级字体栈（-apple-system / PingFang SC / Microsoft YaHei）

## 使用方式

直接用浏览器打开：

```
travel-planner-mockup.html
```

## 适用场景

- AI Agent 类产品的界面设计参考
- 前端切图 / 组件化开发的视觉基准
- 产品演示与需求评审
